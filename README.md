## Set Up

### Install

Download the repo. 

```bash
git clone https://github.com/teamallnighter/daves-master-extracts.git
```

Make sure you have hugo set up 

#### Init Netlify

```bash
ntl init
```

#### Serve Locally

```bash 
hugo server
```

#### Deploy

```bash
hugo -D ntl deploy --prod
```
